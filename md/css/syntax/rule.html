# 样式表规则

# At 规则

MDN 描述

一个 at-rule 是一个 CSS 语句，以 @ 符号开头，后跟一个标识符，并包括直到下一个分号的所有内容，; 或下一个 CSS 块，以先到者为准。

# 常规规则

# @charset

@charset 指定样式表中使用的字符编码。它必须是样式表中的第一个元素，而前面不得有任何字符。

@charset "utf-8";
@charset "iso-8859-15";

# @import

@import 用于引入一个 CSS 文件，除了 @charset 规则不会被引入，@import 可以引入另一个文件的全部内容。

@import "mystyle.css";
@import url("mystyle.css");

# @namespace

@namespace 用于跟 XML 命名空间配合的一个规则，表示内部的 CSS 选择器全带上特定命名空间。

@namespace url(http://www.w3.org/1999/xhtml);
@namespace svg url(http://www.w3.org/2000/svg);

# 嵌套规则

# @media

@media 是 media query 使用的规则，它能够对设备的类型进行一些判断。

@media screen and (mid-width: 900px) {
  body {
    font-size: 10pt;
  }
}

# @page

@page 用于分页媒体访问网页时（如定义打印文档时页面）的表现设置，页面是一种特殊的盒模型结构，除了页面本身，还可以设置它周围的盒。

@page {
  size: 8.5in 11in;
  margin: 10%;
  
  @top-left {
    content: "Hamlet";
  }
  @top-right {
    content: "Page " counter(page);
  }
}

# @font-face

@font-face 用于定义一种字体，字体能从远程服务器或者用户本地安装的字体加载。

@font-face {
  font-family: Gentium;
  src: url(http://example.com/fonts/Gentium.woff);
}

# @keyframes

@keyframes 产生一种数据，用于定义动画关键帧。

@keyframes diagonal-slide {
  from {
    left: 0;
    top: 0;
  }
  to {
    left: 100px;
    top: 100px;
  }
}

# @supports

@supports 用于特性检查，它与 @media 比较类似。

@supports (display: flex) {
  .module {
    display: flex;
  }
}

# @ducument

@document 根据文档的 URL 限制其中包含的样式格则。它主要是为用户定义的样式表设计的。

@document url("https://www.example.com/") {
  h1 {
    color: green;
  }
}

# 普通规则

# 简单选择器

# 全体选择器

全体选择器 * 可以选中任意元素。

* {
  ...
}

# 类型选择器

类型选择器根据一个元素的标签名来选中元素。

div {
  ...
}

类型选择器看上去非常简单，但实际上，还必须要考虑 HTML 或者 XML 元素的命名空间问题。

# id 选择器

id 选择器 # 是针对特性属性 id 的选择器。

#myid {
  ...
}

# class 选择器

class 选择器 . 是针对特定属性 class 的选择器。

.classa {
  ...
}

# 属性选择器

属性选择器根据 HTML 元素来选中元素。属性选择器形态：

形态 说明
[attr] 表示带有以 attr 命名的属性的元素
[attr=value] 表示带有以 attr 命名的属性，且属性值为 value 的元素
[attr~=value] 表示带有以 attr 命名的属性的元素，并且该属性是一个以空格作为分隔的值列表，其中至少一个值匹配 value
[attr|=value] 表示带有以 attr 命名的属性的元素，属性值为 value 或是以 value- 为前缀开头
[attr^=value] 表示带有以 attr 命名的属性，且属性值是以 value 开头的元素
[attr$=value] 表示带有以 attr 命名的属性，且属性值是以 value 结尾的元素
[attr*=value] 表示带有以 attr 命名的属性，且属性值包含有 value 的元素

# 组合选择器

# 相邻兄弟选择器

相邻兄弟选择器 + 介于两个选择器之间，当第二个元素紧跟在第一个元素之后，并且两个元素都是属于同一个父元素的子元素，则第二个元素将被选中。

/* 图片后面紧跟着的段落将被选中 */
img + p {
  font-style: bold;
}

# 普通兄弟选择器

兄弟选择符，位置无需紧邻，只需同层级，A ~ B 选中 A 元素之后所有同层级 B 元素。

p ~ span {
  color: red;
}

# 子选择器

当使用 > 选择符分隔两个元素时，它只会匹配那些作为第一个元素的直接后代（子元素）的第二个元素。

/* 选中 div 中的所有 span 直接后代 */
div > span {
  background-color: red;
}

# 后代选择器

当使用 [space] 空格选择符连接两个元素时，该选择器可以只匹配那些由第一个元素作为祖先元素的所有第二个元素（后代元素）。

/* 选中 div 中的所有 span 节点（包括嵌套节点） */
div span {
  background-color: red;
}

# 伪类选择器

伪类选择器是一系列由 CSS 规定好的选择器，它们以 : 开头。伪类选择器由普通型和函数型两种。

具体参考：CSS 伪类 - MDN

# 伪元素选择器

伪元素本身不单单是一种选择规则，它还是一种机制。一个选择器中只能使用一个伪元素，伪元素必须紧跟在语句的简单选择器 / 基础选择器之后。

# ::first-line

::first-line 在某块级元素的第一行应用样式。第一行的长度取决于很多因素，包括元素宽度，文档宽度和文本的文字大小。

# ::first-letter

::first-letter 会选中某块级元素第一行的第一个字母，并且文字所处的行之前没有其他内容（如图片和内联的表格）。

::first-line 和 ::first-letter 只能在一个 display 值为 block、inline-block、table-cell 或者 table-caption 中有用。在其他的类型中是不起作用的。

# ::before

::before 创建一个伪元素，其将成为匹配选中的元素的第一个子元素。常通过 content 属性来为一个元素添加修饰性的内容。此元素默认为行内元素。

# ::after

::after 创建一个伪元素，其将成为匹配选中的元素的最后一个子元素。常通过 content 属性来为一个元素添加修饰性的内容。此元素默认为行内元素。
