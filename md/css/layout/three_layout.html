# 两侧定宽，中间自适应

# float

float 元素是有浮动范围的，在它的包含块中，如果它之前有内容，会阻止它向上浮动，因此在不要求 content 元素首先渲染的情况下才会用浮动布局，因为此时 content 元素必须位于浮动元素下方。

<div class="layout">
  <header class="header">header</header>
  <aside class="aside-left">aside-left</aside>
  <aside class="aside-right">aside-right</aside>
  <main class="main">main</main>
  <footer class="footer">footer</footer>
</div>

body {
  margin: 0;
  padding: 0;
}

html,
body,
.layout {
  height: 100%;
}

.header,
.footer {
  height: 50px;
  text-align: center;
  line-height: 50px;
  background-color: burlywood;
}

.aside-left,
.aside-right {
  width: 200px;
  height: calc(100% - 100px);
  text-align: center;
  background-color: aquamarine;
}

.aside-left {
  float: left;
}

.aside-right {
  float: right;
}

.main {
  overflow: hidden;
  height: calc(100% - 100px);
  text-align: center;
  background-color: #eee;
}

# 圣杯布局

# 圣杯 DOM 结构

<div class="header">#header</div>
<div class="container">
  <div class="center">#center</div>
  <div class="left">#left</div>
  <div class="right">#right</div>
</div>
<div class="footer">#footer</div>

首先定义出整个布局的 DOM 结构，主体部分是由 container 包裹的 center、left、right 三列，其中 center 定义在最前面。

# 圣杯 CSS 代码

假设左侧的固定宽度为 200px，右侧的固定宽度为 150px，则首先在 container 上设置：

.container {
  padding-left: 200px;
  padding-right: 150px;
}

为左右两列预留出相应的空间，得到如下的示意图：

随后分别为三列设置宽度与浮动，同时对 footer 清除浮动：

.center {
  float: left;
  width: 100%;
}

.left {
  float: left;
  width: 200px;
}

.right {
  float: left;
  width: 150px;
}

.footer {
  clear: both;
}

得到如下效果：

根据浮动的特性，由于 center 的宽度为 100%，即占据了第一行的所有空间，所以 left 和 right 被“挤”到了第二行。

接下来的工作是将 left 放置到之前预留出的位置上，这里使用负外边距 (nagetive margin)：

.left {
  margin-left: -100%;
  width: 200px;
}

随后还需要使用定位 position 的方法：

.left {
  position: relative;
  margin-left: -100%;
  width: 200px;
  right: 200px;
}

这里使用 position: relative 和 right: 200px 将 left 的位置在原有位置基础上左移 200px，以完成 left 的放置：

接下来放置 right，只需添加一条声明即可：

.right {
  margin-right: -150px;
  width: 150px;
}

得到最终的效果图：

至此，布局效果完成。不过还需要考虑最后一步，那就是页面的最小宽度：想要保证该布局效果正常显示，由于两侧都具有固定的宽度，所以需要给定页面一个最小的宽度，但这并不知识简单的 200 + 150 = 350px。

回想之前 left 使用了 position: relative，所以就意味着在 center 开始的区域，还存在着一个 left 的宽度。所以页面的最小宽度应该设置为 200 + 150 + 200 = 550px。

body {
  min-width: 550px;
}

最终 CSS 代码：

body {
  margin: 0;
  padding: 0;
  min-width: 550px;
}

.header,
.footer {
  height: 8vh;
  line-height: 8vh;
  text-align: center;
  color: white;
  background-color: #616161;
}

.container {
  padding-left: 200px;
  padding-right: 150px;
  height: 84vh;
  line-height: 84vh;
  text-align: center;
}

.center {
  float: left;
  width: 100%;
  background-color: #9E9E9E;
}

.left {
  position: relative;
  float: left;
  margin-left: -100%;
  width: 200px;
  right: 200px;
  background-color: #039BE5;
}

.right {
  float: left;
  margin-right: -150px;
  width: 150px;
  background-color: #FF5722;
}

.footer {
  clear: both;
}

# 双飞翼布局

# 双飞翼 DOM 结构

<div class="header">#header</div>
<div class="container">
  <div class="center">#center</div>
  <div class="left">#left</div>
  <div class="right">#right</div>
</div>
<div class="footer">#footer</div>

双飞翼布局的 DOM 结构与圣杯布局的区别是用 container 仅包裹住 center。

# 双飞翼 CSS 代码

按照与圣杯布局相同的思路，首先设置各列的宽度与浮动，并且为左右两列预留出空间，以及为 footer 设置浮动清除：

.container {
  float: left;
  width: 100%;
}

.center {
  margin-left: 200px;
  margin-right: 150px;
}

.left {
  float: left;
  width: 200px;
}

.right {
  float: left;
  width: 150px;
}

.footer {
  clear: both;
}

以上代码将 container、left、right 设置 float: left，而在 container 内部，center由于没有设置浮动，所以其宽度默认为 container 的100%，通过对其设置 margin-left 和 margin-right 为左右两列预留出了空间。

将 left 放置到预留位置：

.left {
  width: 200px;
  margin-left: -100%;
}

将 right 放置到预留位置：

.right {
  width: 150px;
  margin-left: -150px;
}

最后计算最小页面宽度：由于双飞翼布局没有用到 position: relative 进行定位，所以最小页面宽度应该为 200 + 150 = 350px。

但是当页面宽度缩小到 350px 附近时，会挤占中间栏的宽度，使得其内容右侧栏被覆盖，如下所示：

因此在设置最小页面宽度时，应该适当增加一些宽度以供中间栏使用（假设为150 px），则有：

body {
  min-width: 500px;
}

至此，双飞翼布局完整的 CSS 代码：

body {
  margin: 0;
  padding: 0;
  min-width: 500px;
}

.header,
.footer {
  height: 8vh;
  line-height: 8vh;
  text-align: center;
  color: white;
  background-color: #616161;
}

.left,
.right {
  float: left;
  height: 84vh;
  line-height: 84vh;
  text-align: center;
}

.container {
  float: left;
  width: 100%;
  height: 84vh;
  line-height: 84vh;
  text-align: center;
}

.center {
  padding-left: 200px;
  padding-right: 150px;
  background-color: #9E9E9E;
}

.left {
  margin-left: -100%;
  width: 200px;
  background-color: #039BE5;
}

.right {
  margin-left: -150px;
  width: 150px;
  background-color: #FF5722;
}

.footer {
  clear: both;
}
