# 1.排序算法

# 排序

# 定义

稳定排序：如果 a 原本在 b 前面，而 a = b，排序之后 a 仍然在 b 的前面。

不稳定排序：如果 a 原本在 b 前面，而 a = b，排序之后 a 可能出现在 b 的后面。

内部排序：所有排序操作都在内存中完成。

外部排序：由于数据太大，因此把数据放在磁盘中，而排序通过磁盘和内存的的数据传输才能进行。

# 排序分类

  1. 插入、冒泡排序的速度较慢，但参加排序的序列局部或整体有序时，这种排序能达到较快的速度
  2. 当 n 较小时，对稳定性不作要求时宜用选择排序，对稳定性有要求时宜用插入或冒泡排序
  3. 当 n 较大时，关键字元素比较随机，对稳定性没要求宜用快速排序
  4. 当 n 较大时，关键字元素可能出现本身是有序的，对稳定性有要求时，空间允许的情况下宜用归并排序，对稳定性没有要求时宜用堆排序

# 冒泡排序

算法描述

冒泡排序 (Bullle Sort) 是一种简单的排序算法。它重复地走放过要排序的序列，一次比较两个元素，如果它们的顺序错误就把它们交换过来。走访序列的工作是重复地进行直到没有再需要交换，也就是说该序列已经排序完成。

算法步骤

  1. 比较相邻的元素。如果第一个比第二个大，就交换它们两个
  2. 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对，这样在最后的元素应该会是最大的数
  3. 针对所有的元素重复以上的步骤，除了最后一个
  4. 重复步骤 1 - 3，直到排序完成

算法分析

  • 最佳情况：T(n) = O(n)

    当输入的序列已经是正序时，无需排序

  • 最差情况：T（n) = O(n^2)

    当输入的序列是反序时，每一步都需要交换操作

  • 平均情况：T(n) = O(n^2)

JavaScript 实现

function bubbleSort(arr) {
  for (let i = 0, len = arr.length; i < len; i++) {
    for (let j = 0; j < len - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        [ arr[j], arr[j + 1] ] = [ arr[j + 1], arr[j] ]
      }
    }
  }
  return arr
}

const arr = [4, 6, 6, 8, 4, 4, 9, 4, 7, 1]
console.log(bubbleSort(arr))  // -> [1, 4, 4, 4, 4, 6, 6, 7, 8, 9]

# 缓存 pos

设置一标志变量 pos，用于记录每趟排序中最后一次进行交换的位置。由于 pos 位置之后的记录均已交换到位，故在进行下一趟排序时只要扫描到 pos 位置即可。

function bubbleSort2(arr) {
  let i = arr.length - 1
  while (i > 0) {
    let pos = 0   // 每趟开始时, 无记录交换
    for (let j = 0; j < i; j++) {
      if (arr[j] > arr[j + 1]) {
        pos = j;  // 记录交换的位置
        [ arr[j], arr[j + 1] ] = [ arr[j + 1], arr[j] ]
      }
    }
    i = pos       // 为下一趟排序作准备
  }
  return arr
}

const arr = [4, 6, 6, 8, 4, 4, 9, 4, 7, 1]
console.log(bubbleSort2(arr))  // -> [1, 4, 4, 4, 4, 6, 6, 7, 8, 9]

# 双向遍历

传统冒泡排序中每一天排序操作只能找到一个最大值和最小值。通过在每趟排序中进行正向和反向两遍冒泡，一次可以得到两个最终值（最大者和最小者），从而使排序躺数几乎减少了一半。

function bubbleSort3(arr) {
  let low = 0, high = arr.length - 1
  while (low < high) {
    // 正向冒泡, 找到最大者
    for (let i = low; i < high; i++) {
      if (arr[i] > arr[i + 1]) {
        [ arr[i], arr[i + 1] ] = [ arr[i + 1], arr[i] ]
      }
    }
    --high  // 修改 high 值, 前移一位
    // 反向冒泡, 找到最小者
    for (let i = high; i > low; i--) {
      if (arr[i - 1] > arr[i]) {
        [ arr[i - 1], arr[i] ] = [ arr[i], arr[i - 1] ]
      }
    }
    ++low   // 修改 low 值, 后移一位
  }
  return arr
}

const arr = [4, 6, 6, 8, 4, 4, 9, 4, 7, 1]
console.log(bubbleSort3(arr))  // -> [1, 4, 4, 4, 4, 6, 6, 7, 8, 9]

# 双向遍历 + 缓存 pos

结合上述两种方法。

function bubbleSort4(arr) {
  let low = 0, high = arr.length - 1
  while (low < high) {
    let lowPos = 0, highPos = 0
    // 正向冒泡, 找到最大者, 并记录交换的位置
    for (let i = low; i < high; i++) {
      if (arr[i] > arr[i + 1]) {
        [ arr[i], arr[i + 1] ] = [ arr[i + 1], arr[i] ]
        highPos = i
      }
    }
    high = highPos  // 修改 high 值为交换的位置
    // 反向冒泡, 找到最小者, 并记录交换的位置
    for (let i = high; i > low; i--) {
      if (arr[i - 1] > arr[i]) {
        [ arr[i - 1], arr[i] ] = [ arr[i], arr[i - 1] ]
        lowPos = i
      }
    }
    low = lowPos  // 修改 low 值为交换的位置
  }
  return arr
}

const arr = [4, 6, 6, 8, 4, 4, 9, 4, 7, 1]
console.log(bubbleSort4(arr))  // -> [1, 4, 4, 4, 4, 6, 6, 7, 8, 9]

# 选择排序

算法描述

选择排序 (Selection Sort) 是一种简单直观的排序算法，无论什么数据进去都是 O(n2) 的时间复杂度。因此在选择使用它时，数据规模越小越好。

它的工作原理：首先在未排序序列中找到最小（大）的元素，存放到排序序列的起始位置，然后，再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。以此类推，直到所有元素均排序完毕。

算法步骤

n 个记录的直接选择排序可经过 n - 1 趟直接选择排序得到有序结果：

  1. 初始状态：无序区为 R[1...n]，有序区为空
  2. 第 i 趟排序 (i = 1,2,3,...,n-1) 开始时，当前有序区和无序区分别为 R[1...i-1] 和 R[i...n]。该趟排序从当前无序区中选出关键字最小的记录 R[k]，将它和无序区的第 1 个记录 R 交换，使 R[1...i] 和 R[i+1...n] 分别变为记录个数增加 1 的新有序区和记录个数减少 1 的新无序区
  3. n - 1 趟排序结束，数组有序化

算法分析

  • 最佳情况：T(n) = O(n^2)
  • 最差情况：T(n) = O(n^2)
  • 平均情况：T(n) = O(n^2)

JavaScript 实现

function selectionSort(arr) {
  for (let i = 0, len = arr.length; i < len - 1; i++) {
    let minIndex = i
    for (let j = i + 1; j < len; j++) {
      if (arr[j] < arr[minIndex]) {
        minIndex = j       // 保存最小值的索引
      }
    }
    if (minIndex !== i) {  // 索引有更新时才进行交换
      [ arr[i], arr[minIndex] ] = [ arr[minIndex], arr[i]]
    }
  }
  return arr
}

const arr = [7, 5, 5, 7, 8, 9, 9, 6, 3, 6]
console.log(selectionSort(arr))  // [3, 5, 5, 6, 6, 7, 7, 8, 9, 9]

# 插入排序

算法描述

插入排序 (Insertion Sort) 是一种简单直观的排序算法。它的工作原理是通过构造有序序列，对于未排序序列，在已排序序列中从后向前扫描，找到相应位置并插入。

插入排序在实现上，通常采用 in-place 排序（即只需要用到 O(1) 的额外空间的排序），因而在从后向前扫描过程中，需要反复把已排序元素逐步向后挪位，为最新元素提供插入空间。

算法步骤

  1. 从第一个元素开始，该元素可以认为已经被排序
  2. 取出下一个元素，在已排序序列中从后向前扫描
  3. 如果该元素（已排序）大于新元素，将该元素移到下一位置
  4. 重复步骤 3，直到找到已排序元素小于或者等于新元素的位置
  5. 将新元素插入到该位置
  6. 重复步骤 2 - 5

算法分析

  • 最佳情况：T(n) = O(n)，输入数组按升序排序
  • 最差情况：T(n) = O(n^2)，输入数组按降序排序
  • 平均情况：T(n) = O(n^2)

JavaScript 实现

function insertionSort(arr) {
  for (let i = 1, len = arr.length; i < len; i++) {
    let key = arr[i], j = i - 1
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j--]
    }
    arr[j + 1] = key
  }
  return arr
}

const arr = [3, 6, 1, 9, 9, 9, 3, 1, 7, 8]
console.log(insertionSort(arr))  // [1, 1, 3, 3, 6, 7, 8, 9, 9, 9]

# 二分查找方式

查找插入位置时使用二分查找的方式。

function insertionSort2(arr) {
  for (let i = 1, len = arr.length; i < len; i++) {
    let key = arr[i], left = 0, right = i - 1
    while (left <= right) {
      const middle = parseInt((left + right) / 2)
      key < arr[middle] ? right = middle - 1 : left = middle + 1
    }
    for (let j = i - 1; j >= left; j--) {
      arr[j + 1] = arr[j]
    }
    arr[left] = key
  }
  return arr
}

const arr = [3, 6, 1, 9, 9, 9, 3, 1, 7, 8]
console.log(insertionSort(arr))  // [1, 1, 3, 3, 6, 7, 8, 9, 9, 9]

# 希尔排序

算法描述

希尔排序 (Shell Sort) 也称递减增量排序，是插入排序的一种更高效的改进版本。

希尔排序是基于插入排序的以下两点性质而提出改进方法的：

  • 插入排序在对几乎已经排好序的数据操作时，效率高，即可以达到线性排序的效率
  • 插入排序一般来说是低效的，因为每次只能将数据移动一位

希尔排序的核心在于间隔序列的设定。既可以提前设定好间隔序列，也可以动态的定义间隔序列。

算法步骤

先将整个待排序的记录序列分割成为若干个子序列分别进行插入排序，具体算法步骤：

  1. 选择一个增量序列 t1,t2,...,tk，其中 ti > tj, tk = 1
  2. 按增量序列个数 k，对序列进行 k 趟排序
  3. 每趟排序，根据对应的增量 ti，将待排序列分割成若干长度为 m 的子序列，分别对各子序列进行直接插入排序。仅增量因子为 1 时，整个序列作为一个表来处理，表长度即为整个序列的长度

算法分析

  • 最佳情况：T(n) = O(nlog2n)
  • 最差情况：T(n) = O(nlog2n)
  • 平均情况：T(n) = O(nlogn)

JavaScript 实现

function shellSort(arr) {
  const len = arr.length
  let gap = 1
  while (gap < len / 5) {
    gap = gap * 5 + 1
  }
  while(gap > 0) {
    // 注意下面这段 for 循环和插入排序极为相似
    for (let i = gap; i < len; i++) {
      let key = arr[i], j = i - gap
      while (j >= 0 && arr[j] > key) {
        arr[j + gap] = arr[j]
        j -= gap
      }
      arr[j + gap] = key
    }
    gap = Math.floor(gap / 2)
  }
  return arr
}

const arr = [4, 4, 9, 9, 3, 9, 4, 1, 4, 5]
console.log(shellSort(arr))  // [1, 3, 4, 4, 4, 4, 5, 9, 9, 9]

# 归并排序

算法描述

归并排序 (Merge Sort) 是建立在归并操作上的一种有效的排序算法。该算法是采用分治法的一个非常典型的应用。

归并排序是一种稳定的排序方法。将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序。若将两个有序表合并成一个有序表，称为2-路归并。

算法步骤

  1. 把长度为 n 的输入序列分成两个长度为 n / 2 的子序列
  2. 对这两个子序列分别采用归并排序
  3. 将两个排序好的子序列合并成一个最终的排序序列

算法分析

和选择排序一样，归并排序的性能不受输入数据的影响，但表现比选择排序排序好得多，因为始终都是 O(nlogn) 的时间复杂度，代价是需要额外的内存空间。

  • 最佳情况：T(n) = O(n)
  • 最差情况：T(n) = O(nlogn)
  • 平均情况：T(n) = O(nlogn)

JavaScript 实现

function mergeSort(arr) {
  const len = arr.length
  if (len < 2) {
    return arr
  }
  const middle = Math.floor(len / 2),
        left = arr.splice(0, middle),
        right = arr
  return merge(mergeSort(left), mergeSort(right))
}

function merge(left, right) {
  const result = []
  while (left.length && right.length) {
    left[0] <= right[0] ? result.push(left.shift()) : result.push(right.shift())
  }
  return result.concat(left, right)
}

const arr = [43, 29, 4, 1, 14, 47, 23, 42, 18, 9]
console.log(mergeSort(arr))  // [1,  4,  9, 14, 18, 23, 29, 42, 43, 47]

# 快速排序

算法描述

快速排序 (Quick Sort) 由于排序效率在同为 O(nlogn) 的几种排序方法中效率较高，因此经常被采用。基于分治法的思想，快速排序是一种既不浪费空间又可以快一点的排序算法。

快速排序的基本思想：通过一趟排序将待排记录分隔成独立的两部分，其中一部分记录的关键字均比另一部的关键字小，则可分别对这两部分记录继续进行排序，以达到整个序列有序。

算法步骤

  1. 从序列中挑出一个元素，称为基准 (pivot)

  2. 重新排序序列，所有元素比基准值小的摆放在基准前面，所有元素比基准值大的摆放在基准的后面（相同的数可以到任一边）

    在这个分区退出之后，该基准就处于序列的中间位置。这个称为分区 (partition) 操作

  3. 递归地 (recursive) 把小于基准值元素的子序列和大于基准值元素的子序列排序

算法分析

  • 最佳情况：T(n) = O(nlogn)
  • 最差情况：T(n) = O(n^2)
  • 平均情况：T(n) = O(nlogn)

JavaScript 实现

// ES6 版本
function quickSort(arr) {
  if (arr.length <= 1) {
    return arr
  }
　const pivotIndex = Math.floor(arr.length / 2),  // 基准索引
        pivot = arr.splice(pivotIndex, 1)[0],     // 基准值  
        left = [], right = []
  for (const item of arr) {
    item < pivot ? left.push(item) : right.push(item)
  }
　return quickSort(left).concat([pivot], quickSort(right))
}

const arr = [4, 2, 8, 8, 6, 9, 7, 4, 4, 2]
console.log(quickSort(arr))  // [2, 2, 4, 4, 4, 6, 7, 8, 8, 9]

// In-place 版本
function quickSort(arr, low, high) {
  if (low < high) {
    let pivot = partition(arr, low, high)
    quickSort(arr, low, pivot - 1)
    quickSort(arr, pivot + 1, high)
  }
  return arr
}

function partition(arr, low, high) {
  let pivotkey = arr[low]
  while (low < high) {
    while (high > low && arr[high] >= pivotkey) {
      high--
    }
    arr[low] = arr[high]
    while (high > low && arr[low] <= pivotkey) {
      low++
    }
    arr[high] = arr[low]
  }
  arr[low] = pivotkey
  return low
}

# 堆排序

算法描述

堆排序 (Heap Sort) 是指利用堆这种数据结构所设计的一种排序算法。堆积是一个近似完全二叉树的结构，并同时满足堆积的性质：即子节点的键值或索引总是小于（或者大于）它的父节点。

算法步骤

  1. 将初始待排序关键字序列 (R1,R2,...,Rn) 构建成大顶堆，此堆为初始的无序区
  2. 将堆顶元素 R[1] 与最后一个元素 R[n] 交换，此时得到新的无序区 (R1,R2,...,Rn-1) 和新的有序区 (Rn)，且满足 R[1,2,...,n-1] <= R[n]
  3. 由于交换后新的堆顶 R[1] 可能违反堆的性质，因此需要对当前无序区 (R1,R2,...Rn-1) 调整为新堆，然后再次将 R[1] 与无序区最后一个元素交换，得到新的无序区 (R1,R2,...Rn-2) 和新的有序区 (Rn-1, Rn)
  4. 不断重复步骤 3 直到有序区的元素个数为 n-1，则整个排序过程完成

算法分析

  • 最佳情况：T(n) = O(nlogn)
  • 最差情况：T(n) = O(nlogn)
  • 平均情况：T(n) = O(nlogn)

JavaScript 代码实现

function heapSort(arr) {
  const size = arr.length
  // 初始化堆, i 从最后一个父节点开始调整, 直到节点均调整完毕
  for (let i = Math.floor(size / 2) - 1; i >= 0; i--) {
    heapify(i, size)
  }
  // 堆排序: 先将第一个元素和已拍好元素前一位作交换, 再重新调整, 直到排序完毕
  for (let i = size - 1; i > 0; i--) {
    [ arr[0], arr[i] ] = [ arr[i], arr[0] ]
    heapify(0, i)
  }

  return arr
}

function heapify(start, end) {
  let dad = start, son = dad * 2 + 1
  if (son >= end) {
    return
  }
  if (son + 1 < end && arr[son] < arr[son + 1]) {
    son += 1
  }
  if (arr[dad] <= arr[son]) {
    [ arr[dad], arr[son] ] = [ arr[son], arr[dad] ]
    heapify(son, end)
  }
}

const arr = [2, 6, 7, 4, 4, 8, 6, 7, 3, 8]
console.log(heapSort(arr))  // [2, 3, 4, 4, 6, 6, 7, 7, 8, 8]

# 计数排序

算法描述

计数排序 (Counting Sort) 是一种稳定的排序算法。计数排序使用一个额外的数组 counts，其中第 i 个元素是待排序数组 array 中值等于 i 的元素的个数。然后根据数组 counts 来将 array 中的元素排到正确的位置。它只能对整数进行排序。

计数排序的核心在于将输入的数据值转化为键存储在额外开辟的数组空间中。

算法步骤

  1. 找出待排序的数组中最大和最小的元素
  2. 统计数组中每个值为 i 的元素出现的次数，存入数组 counts 的第 i 项
  3. 对所有的计数累加（从 counts 中的第一个元素开始，每一项和前一项相加）
  4. 反向填充目标数组：将每个元素 i 放在新数组 sorted 第 counts[i] 项，每放一个元素就将 counts[i] 减去 1

算法分析

当输入的元素是 n 个 0 到 k 之间的整数时，它的运行时间是 O(n + k)。计数排序不是比较排序，排序的速度快于任何比较排序算法。

由于用来计数的数组 counts 的长度取决于待排序数组中数据的范围（等于待排序数组的最大值和最小值的差加上 1），这使得计数排序对于数据范围很大的数组，需要大量时间和内存。

  • 最佳情况：T(n) = O(n + k)
  • 最差情况：T(n) = O(n + k)
  • 平均情况：T(n) = O(n + k)

JavaScript 实现

function countingSort(arr) {
  const counts = [], sorted = []
  let min = arr[0], max = arr[0]
  for(const item of arr) {
    min = min <= item ? min : item
    max = max >= item ? max : item
    counts[item] = counts[item] ? counts[item] + 1 : 1
  }
  while (min < max) {
    counts[min + 1] = (counts[min + 1] || 0) + (counts[min] || 0)
    min++
  }
  for (let i = arr.length - 1; i >= 0; i--) {
    sorted[counts[arr[i]] - 1] = arr[i]
    counts[arr[i]]--
  }
  return sorted
}
