# 如何遍历一棵树

  • 广度优先遍历 (BFS)

    按照高度顺序一层一层的访问整棵树，高层次的节点将会比低层次的节点先被访问到。

  • 深度优先遍历 (DFS)

    采用深度作为优先级，以便从根开始一直到达某个确定的叶子，然后再返回根到另一个分支。

    深度优先搜索策略又可以根据根节点、左孩子和右孩子的相对顺序被细分为前序遍历、中序遍历和后序遍历。

# #144 二叉树的前序遍历

前序遍历的顺序特点适合使用 DFS，使用辅助栈 stack 存储当前处理节点， 每次都优先处理左子树。

/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var preorderTraversal = function(root) {
  if (!root) return []
  
  const res = [], stack = [root]
  
  while (stack.length) {
    root = stack.pop()
    res.push(root.val)

    // 栈结构的特点: 先进后出，优先处理左子树节点
    if (root.right) stack.push(root.right)
    if (root.left) stack.push(root.left)
  }
  
  return res
};

# #94 二叉树的中序遍历

中序遍历的顺序特点适合使用 DFS，与前序遍历的区别在于中序遍历每次都优先处理左子树。

/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var inorderTraversal = function(root) {
  const res = [], stack = []
  
  while (root || stack.length) {
    // 找到最左子树
    while (root) {
      stack.push(root.val)
      root = root.left
    }

    // 左子树出栈后，栈顶元素为当前子树的根节点
    root = stack.pop()
    res.push(root.val)
    root = root.right
  }
};

# #145 二叉树的后序遍历

从根节点开始依次迭代，弹出栈顶元素输出到输出列表中，然后依次压入它的所有孩子节点，按照从上到下、从左至右的顺序依次压入栈中。

因为深度优先搜索后序遍历的顺序是从下到上、从左至右，所以需要将输出列表逆序输出。

/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var postorderTraversal = function(root) {
  if (!root) return []
  
  const res = [], stack = [root]
  
  while (stack.length) {
    root = stack.pop()
    res.push(root.val)

    if (root.left) stack.push(root.left)
    if (root.right) stack.push(root.right)
  }
  return res.reverse()
};

# #105 从前序和中序遍历序列构造对应二叉树

前序遍历的顺序是 Root -> Left -> Right，这就能方便的从根开始构造一棵树。

首先，preorder 中的第一个元素一定是树的根，这个根又将 inorder 序列分成了左右两棵子树。现在只需要将前序遍历的数组中删除根元素，然后重复上面的过程处理左右两棵子树。

/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {number[]} preorder
 * @param {number[]} inorder
 * @return {TreeNode}
 */
var buildTree = function(preorder, inorder) {
  if (!preorder.length)
    return null
  
  // 前序遍历第一个值为根节点
  const root = TreeNode(preorder[0])
  // 因为没有重复元素，所以可以直接根据值来查找根节点在中序遍历中的位置
  let mid = inorder.indexOf(preorder[0])
  // 构造左子树
  root.left = buildTree(preorder.slice(1, mid + 1), inorder.slice(0, mid))
  // 构造右子树
  root.right = buildTree(preorder.slice(mid + 1), inorder.slice(mid + 1))
  
  return root
};

# #106 从中序与后序遍历序列构造二叉树

后序遍历的顺序是 Left -> Right -> Root，这就能方便的从根开始构造一棵树。

首先，postorder 中的最后一个元素一定是树的根，这个根又将 inorder 序列分成了左右两棵子树。现在只需要将后序遍历的数组中删除根元素，然后重复上面的过程处理左右两棵子树。

/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {number[]} inorder
 * @param {number[]} postorder
 * @return {TreeNode}
 */
var buildTree = function(inorder, postorder) {
  if (!postorder.length)
    return null
  
  let root_idx = postorder.length - 1
  // 后序遍历的最后一个值为根节点
  const root = new TreeNode(postorder[root_idx])
  // 因为没有重复元素，所以可以直接根据值来查找根节点在中序遍历中的位置
  let mid = inorder.indexOf(postorder[root_idx])
  // 构造左子树
  root.left = buildTree(inorder.slice(0, mid), postorder.slice(0, mid))
  // 构造右子树
  root.right = buildTree(inorder.slice(mid + 1), postorder.slice(mid, root_idx))
  
  return root
};
